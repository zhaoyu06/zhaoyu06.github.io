取整方法总结

1.直接赋值给整数变量
int i = 3.5; 或 i = (int) 3.5;
2.整数除法运算符‘ / ’取整
‘ / ’本身就有取整功能(int / int)，可是整数除法对负数的取整结果和使用的C编译器有关。
3.使用floor函数
floor(x)返回的是小于或等于x的最大整数。
如：
floor(3.5) = 3
floor(-3.5) = -4
4、使用ceil函数
ceil(x)返回的是大于x的最小整数。如：

ceil(3.5) = 4

ceil(-3.5) = -3

floor()是向负无穷大舍入，floor(-3.5)  = -4。

ceil()是向正无穷大舍入。ceil(-3.5)  = -3。

可是在C里面ceil和floor()函数是返回double型。
